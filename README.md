# Prueba-de-Unidad-3
El proyecto LogiCodeUTA consiste en un aplicativo interactivo desarrollado en C++ para la carrera de Ingeniería en Software de la Universidad Técnica de Ambato, diseñado por Mateo Alexander Muyulema Moyolema en Ambato, Ecuador, con el objetivo de automatizar el procesamiento de datos académicos y operaciones lógicas. El sistema integra un módulo de aritmética básica con validaciones para evitar errores críticos como la división por cero y un motor de gestión de archivos que permite la lectura automatizada de un listado de hasta 30 estudiantes desde el archivo estudiantes.txt. Mediante estructuras de control y el uso de struct para la organización de datos, el software calcula promedios generales, identifica calificaciones máximas y mínimas, y determina el número de alumnos aprobados y reprobados, exportando finalmente un reporte técnico detallado en el archivo resultados.txt que incluye la fecha de proceso, las estadísticas del curso y la identificación formal del autor.
